# Documentation Versions

DeepChopper maintains versioned documentation to help you find information relevant to your installed version.

## Available Versions

### Latest (Stable)
- **Version**: Latest stable release
- **Recommended for**: Production use
- [View Latest Documentation](https://ylab-hi.github.io/DeepChopper/latest/)

### Development (Dev)
- **Version**: Latest development version
- **Recommended for**: Contributors and early adopters
- [View Dev Documentation](https://ylab-hi.github.io/DeepChopper/dev/)

## Version History

To view documentation for a specific version:

1. Click the version selector in the header
2. Choose your installed version
3. Browse documentation specific to that release

## Installing Specific Versions

```bash
# Install latest stable
pip install deepchopper

# Install specific version
pip install deepchopper==1.2.9

# Install from development
pip install git+https://github.com/ylab-hi/DeepChopper.git
```

## Need Help?

If you're unsure which version to use:

- **For production**: Use the latest stable version
- **For development**: Use the dev version
- **For specific features**: Check the changelog for version differences

See the [Installation Guide](installation.md) for more details.
